## POST /api/chat

Ask the AI assistant about a specific dish.

**Request:**
```json
{
  "prompt": "How spicy is it?",
  "menu_item_id": "pad_thai",
  "session_id": "uuid-string"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `prompt` | string | Customer's question (required) |
| `menu_item_id` | string | Menu item ID for context |
| `session_id` | string | Unique session ID (generated by frontend) |

**Response:**
```json
{
  "answer": "Pad Thai is mildly spicy...",
  "history": [
    { "role": "user", "content": "..." },
    { "role": "assistant", "content": "..." }
  ]
}
```

Notes:
- First message in a session automatically injects dish name context
- History is maintained in-memory (lost on server restart)
- The agent uses Azure AI Foundry index search for dish details

## POST /api/order

Place a food order. Returns immediately; order processing happens in background.

**Request:**
```json
{
  "table": 1,
  "items": [
    {
      "name": "...",
      "qty": 1,
      "price": 70,
      "addons": ["Special (large)", "Extra egg"],
      "note": "no cilantro"
    }
  ],
  "total": 150,
  "note": ""
}
```

| Field | Type | Description |
|-------|------|-------------|
| `table` | int | Table number |
| `items` | array | List of ordered items |
| `items[].name` | string | Dish name (Thai) |
| `items[].qty` | int | Quantity |
| `items[].price` | int | Unit price including add-ons |
| `items[].addons` | string[] | Selected add-on names |
| `items[].note` | string | Per-item note (any language) |
| `total` | int | Grand total in baht |
| `note` | string | Order-level note |

**Response:**
```json
{
  "success": true,
  "message": "Order sent successfully"
}
```

Background processing:
1. Notes are translated to Thai via Azure AI
2. Message is formatted with items, add-ons, notes, and total
3. Formatted message sent to Azure Logic App
4. Logic App forwards to LINE OA

**Error Responses:**

| Status | When |
|--------|------|
| 400 | No items in order, or empty prompt (chat) |
| 404 | Menu item not found (chat) |
| 503 | Azure AI not configured |
| 502 | Azure AI or Logic App request failed |
