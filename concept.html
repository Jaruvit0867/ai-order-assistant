Hackathon Category: Tourism (Travel Tech & Zero-Hardware)

AI Multilingual Order Assistant

ระบบผู้ช่วยรับออเดอร์อัจฉริยะพหุภาษา ทลายข้อจำกัดด้านภาษาของร้านค้ารายย่อย ด้วยสถาปัตยกรรมที่ใช้เพียง "สมาร์ทโฟนและคิวอาร์โค้ด" ประมวลผลผ่าน Azure Cloud 100%

Pain Point

ร้านอาหารระดับชุมชนและสตรีทฟู้ด (Micro-SME) พลาดโอกาสทางเศรษฐกิจจากนักท่องเที่ยวชาวต่างชาติ เนื่องจากข้อจำกัดด้านภาษา

  • ขาดเครื่องมือสื่อสารและอธิบายรายละเอียดอาหารที่ถูกต้องให้ชาวต่างชาติ
  • ต้นทุนระบบการจัดการร้านอาหาร (POS) หรือตู้ Kiosk มีราคาสูงเกินไป
  • นักท่องเที่ยวขาดความมั่นใจในการสั่งอาหารท้องถิ่น (กลัวแพ้อาหาร, สั่งผิด)

Our Solution

ระบบสื่อสารและรับออเดอร์สองทาง ที่ทำงานผ่านสมาร์ทโฟนของผู้ใช้งาน โดยไม่จำเป็นต้องติดตั้งอุปกรณ์เพิ่มเติมที่ร้านค้าเลย

  • แปลและแสดงผลเมนูเป็นภาษาของนักท่องเที่ยวอัตโนมัติ (Auto-Translate)
  • AI Agent ให้ข้อมูลเชิงลึกเกี่ยวกับอาหาร (วัตถุดิบ, รสชาติ) ผ่านเสียงและข้อความ
  • แปลงคำสั่งซื้อเป็น "ภาษาไทย" ส่งตรงเข้ามือถือผู้ขายทันที

System Architecture & Workflow

1. User Interface (Tourist)

ใช้งานผ่านเว็บเบราว์เซอร์บนมือถือนักท่องเที่ยว

Scan & Access: สแกนคิวอาร์โค้ดหน้าร้านเพื่อเปิดเมนู โดยระบบจะแสดงผลเป็นภาษาแม่ของนักท่องเที่ยวโดยอัตโนมัติ

Interactive Q&A: พิมพ์หรือพูดถามรายละเอียดเมนูด้วยภาษาตัวเอง (เช่น มีเนื้อไหม? เผ็ดไหม?) ระบบจะตอบกลับเป็นทั้งข้อความและเสียงพูดอธิบาย

Order Placement: กดยืนยันรายการอาหารที่ต้องการผ่านหน้าจอ

2. Processing (Azure AI)

ศูนย์กลางสมองกลประมวลผลบนคลาวด์

Azure AI Speech

ประมวลผลคำสั่งเสียง (Speech-to-Text) และสร้างเสียงตอบกลับให้นักท่องเที่ยวฟัง (Text-to-Speech)

Azure OpenAI

ทำ RAG ดึงข้อมูลวัตถุดิบเพื่อตอบคำถาม Q&A และสกัดคีย์เวิร์ดรายการอาหารที่สั่ง (Keyword Extraction)

Azure Functions

รันเซอร์วิสแบบ Serverless ส่งข้อมูลคำสั่งซื้อที่แปลเป็น "ภาษาไทย" ไปยังระบบแจ้งเตือน

3. Merchant Notification

รับออเดอร์ผ่านมือถือเครื่องเดิมที่ร้าน

LINE Notify Now

  • ไข่ดาว (1)

ผู้ขายอ่านออเดอร์ภาษาไทย ตักอาหาร และเก็บเงินได้อย่างราบรื่น โดยไม่ต้องพูดภาษาอังกฤษสักคำ!

Business Viability (ความคุ้มค่าและโอกาสขยายผล)

Zero Hardware Barrier

ไม่ต้องลงทุนติดตั้งจอ POS หรืออุปกรณ์ฮาร์ดแวร์ใหม่ ใช้เพียงกระดาษ QR Code 1 แผ่น และสมาร์ทโฟนที่มีอยู่แล้ว ทำให้ร้านค้ารายย่อยตัดสินใจใช้งานได้ทันที (Low Barrier to Entry)

High Cost-Efficiency

สถาปัตยกรรม Serverless และโมเดลภาษาขนาดเล็ก (เช่น GPT-4o-mini) ทำให้ต้นทุนประมวลผลต่อบิลต่ำระดับไม่ถึง 1 บาท คุ้มค่าต่อการทำกำไรและต่อยอดเป็นระบบ Subscription รายเดือน

Highly Scalable

ระบบถูกออกแบบมาให้กลมกลืนกับ Workflow เดิมของร้านอาหารไทย สามารถสเกลขยายให้ครอบคลุมร้านสตรีทฟู้ดทั่วประเทศได้อย่างรวดเร็ว สร้างอิมแพคต่อระบบเศรษฐกิจการท่องเที่ยวฐานราก
